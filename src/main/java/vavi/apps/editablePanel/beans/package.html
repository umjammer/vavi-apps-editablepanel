編集可能パネル{@link vavi.apps.editablePanel.EditablePanel}で使用されるコンポーネントのクラスを提供します．

まだコントローラを java beans 化してないので仕様は決定ではありません． とりあえず新規コンポーネントを作成する際は，

  1. {@link java.awt.Component}(なるべく{@link javax.swing.JComponent}のほうが望ましい)を継承したコンポーネントを作成する．
  2. そのコンポーネントに対するコントローラを，{@link vavi.apps.editablePanel.Controller}を継承して作成する．
  3. そのコンポーネントのプロパティエディタを，{@link vavi.apps.editablePanel.Controller}を implemants して作成する．
  4. vavi/apps/editablePanel/EditablePanel.properties に作成したコントローラを記述しておく． 
    
ep.palette.X.title=your palette title
ep.palette.X.0=your.package.yourComponent1
ep.palette.X.1=your.package.yourComponent2
 :
    Xあなたのコンポーネントを置くパレットのタイトル
    2コンポーネントの順番 (0から7まで)
    your.package.yourComponent#パレットの番号 (0と1はシステムで予約なので2から)
